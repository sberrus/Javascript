Imagenes y Objetos en HTML.

En este apartado entraremos en la parte mas dinamica y visual dentro de html.

Veremos en esta clase los puntos relacionados con la implementacion de imagenes y objetos dentro de html. Para su uso estetico dentro de nuestras paginas webs.
para incluir imagenes dentro de html utilizaremos la etiqueta <img> para introducir las imagenes dentro de nuestro codigo de html.

La etiqueta <img> permite introducir imagenes a nuestras paginas webs mediante enlaces parecidos a como funciona la etiqueta <a>.
La etiqueta <img> tiene ciertos atributos como pueden ser:

src=""
Con este atributo introducimos la URL absoluta o relativa del archivo al cual deseamos implantar dentro de nuestra pagina web. Esta puede utilizar enlaces a archivos locales o a archivos externos que se encuentren en otro servidor.

La diferencia mas significativa entre estas maneras de implantar la imagen seria que en la que utilizamos enlaces externos si en el servidor donde esta alojada la imagen deciden por cualquier razon eliminar la imagen o la foto nosotros tampoco podremos hacer uso de la misma para nuestra pagina web. En cambio si alojamos nosotros mismos la imagen en nuestro servidor, este no se movera a menos que nosotros indiquemos lo contrario.
alt="url"
Este atributo nos permite asignarle una corta descripcion a la imagen que estamos utilizando. Sirve para mejorar el SEO de la pagina web.
longdesc="url"
Con este atributo podemos introducir a la imagen una URL que muestre una descripcion mas detallada de la imagen.
name="texto"
Indica el nombre del elemento imagen.
height= "unidad_de_medida"
Indica la altura del elemento dentro de la pagina.
width= "unidad_de_medida"
Indica el ancho de este elemento dentro de la pagina.

En html los atributos mas utilizados para las imagenes son src que funciona parecido al atributo href de la etiqueta <a> y alt para indicar la descripcion breve de la imagen con una capacidad maxima de 1024 caracteres y es de utilidad para los usuarios con discapacidad visual.
En html podemos introducir imagenes utilizando los medios que tenemos en nuestra propia carpeta de recursos. (Suele llamarse "images" o "imagenes").
Al momento de modificar el tamaño de las imagenes dentro de hmtl con los atributos heigth y width hay que tomar en cuenta que las imagenes deben contener uno solo de los atributos anteriormente mencionados, sino, estos pueden alterara su aspecto y alterar la estetica original de la imagen. Aunque al ser html un lenguaje de marcado de texto, este solo debe asignar la imagen dentro de la pagina web mas no el tamaño de la misma ya que de este proceso ya se encarga CSS que es el encargado total de los aspectos esteticos de las paginas webs.
La etiqueta <img> puede intruducir cualquier elemento grafico dentro del documento html. Pero, se recomienda siempre utilizar los formatos JPG, GIF y PNG. En los navegadores obsoletos pueden haber dificultades a la hora de mostrar archivos de tipo PNG.
