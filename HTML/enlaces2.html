Enlaces 2 en HTML

Los enlaces cuentan con los siguientes Atributos:

name: "texto" - Permite nombrar al enlace para que puedan usarlo otros enlaces.

href: "url" - Indica la URL del recurso enlazado.

hreflang: "codigo_idioma" - Idioma del recurso enlazado.

type: "tipo_de_contenido" - permite "avisar" al navegador sobre le tipo de contenido que se enlaza (imagenes, archivos, etc...) para que pueda prepararse en caso de que no entienda ese contenido.

rel: "tipo_de_relacion" - Describe la relacion del documento actual con el recurso enlazado.

rev: "tipo_de_relacion" - Describe la relacion del recurso enlazado con el documento actual.

charset: "Tipo_de_charset" - Describe la codificacion del recurso enlazado.

Mas Detalles sobre Atributos

Otros tipos de enlaces

script

Etiqueta script:

src: "url" -Indica la direccion del archivo que contiene el codigo. Funciona de igual manera que el atributo "href" de la etiqueta "a"

type: "tipo_de_archivo" - Permite avisar al navegadpr el tipo de codigo que se incluye (Normalmente JavaScript)

defer: "defer" - El codigo no va a modificar el contenido de la pagina web

charset: "tipo_de_charset" - describe la codificacion del codigo enlazado

La etiqueta script se utiliza principalmente para introducir aplicaciones de tipo JavaScript pero tambien permite que dentro de la misma podamos introducir directamente codigo de tipo JavaScript

Link

Etiqueta link

Comparte los mismos atributos que la etiqueta "a":charset, href, hreflag,type,rel,rev

media: "tipo_de_medio" - Indica el medio para el que debe aplicarse la relacion.

Al contrario de java script la etiqueta Link solo puede realizarse dentro de la etiqueta "head."
