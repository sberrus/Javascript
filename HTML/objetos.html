Objetos en HTML

En HTML tambien podemos incluir software mas complejo utilizando aplets de Java o videos en diferentes formatos como QuickTime o Flash. Estos no son arrancados desde el navegador directamente sino que usan peque√±os programas llamados plugins los cuales permite interactuar con ellos.
Para ello utilizamos la etiqueta <object> para imvocar los fragmentos que utilizara luego los diferentes objetos. La etiqueta <object> tiene los siguientes atributos:

  • data = "url" - Indica la URL de los datos que utiliza el objeto.

  • classid, codebase, codetype - Informacion especifica que depende del tipo de objeto.

  • type - indica el tipo de contenido de los datos.

  • height = "unidad_de_medida" - Indica la altura con la que se debe mostrar el objeto.

  • width = "unidad_de_medida" - Indica la anchura con la que se debe mostrar el objeto.

Ademas de usar los objetos. A estos se les pueden asignar parametros mediante el uso de la etiqueta <param name="" value="">. Estas deben estar dentro de la etiqueta <object> y tiene 2 atributos:

  • name = "texto" - Se utiliza para asignarle una identificacion unica al atributo.

  • value = "String" - Se utiliza para asignarle un valor a dicho parametro.

Al incluir videos de tipo flash algunos navegadores tienden a tener ciertos problemas respecto a la rapidez o a que algunos no permitiran ver el video a menos que termine de cargar dicho video un 100%. Para solucionar este problema se introdujo a HTML una etiqueta <embed> la cual permite introducir dichos videos.
El principal inconveniente respecto a usar este metodo es que la pagina web tendra problemas al momento de la validacion de dicha pagina web. Aunque en un futuro se vera que no es completamente necesario que las paginas web tengan precisamente que ser validada ya que incluso la pagina principal de google no esta correctamente validada pero no deja de ser de las paginas mas visitadas del mundo.
