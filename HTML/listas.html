Listas en HTML

Diferentes tipos de listas, etiquetas y atributos.

En HTML contamos principalmente con 3 etiquetas para indicar distintos tipos de listas.

Etiqueta "<ul>"
La etiqueta ul es la etiqueta de lista desordenada. Esta se usa principalmente para hacer listas que no necesiten un orden especifico. Se caracterizan por tener un punto para determinar cada elemento de la lista.
La etiqueta ul debe tener dentro de si la etiqueta <li>, para determinar cada uno de los elementos de la misma. EJ:
  • Esta es una etiqueta <li> dentro de una etiqueta <ul>
Etiqueta "<ol>"
La etiqueta <ol> funciona de manera similar a la etiqueta <ul>, con la diferencia de que en lugar de usar puntos para determinar los elementos de la lista este enumera cada uno de los elementos de la lista empezando desde el 1.
EJ:
  1. Esta es una etiqueta <li> dentro de una etiqueta <ol>
  2. Esta es una etiqueta <li> dentro de una etiqueta <ol>
  3. Esta es una etiqueta <li> dentro de una etiqueta <ol>
Etiqueta "<dl>"
La etiqueta <dl> Se utiliza para hacer listas con definiciones. Esta consta de 2 etiquetas adicionales para determinar sus elementos y sus respectivas definiciones.
La etiqueta <dl> determina la existencia de la lista de definiciones, esta debe contener dentro de si 2 etiquetas siendo <dt> para indicar el elemento de la lista; y <dd> para agregar su respectiva definicion.
EJ:
Este es el elemento a definir Etiqueta <dt>
Esta es la Etiqueta que va a contener la definicion. Etiqueta <dd>
Como bien hemos visto las listas son muy utiles, siempre tomando en cuenta para que las vas a utilizar pueden ser de gran utilidad para tus proyectos y trabajos. Hay es que estar pendiente de agregar los demas atributos como pudieran ser id=<>, entro otros para modificar el aspecto con CSS.
Tambien hay que saber que como bien sabemos HTML sirve principalmente para estructurar la informacion. Lo que son las caracteristicas visuales de la pagina web se establecen mediante las hojas de estilo CSS que veremos proximamente.
