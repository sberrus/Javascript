Mapas de Imagen

Los mapas de imagen se utilizan para incorporar a las imagenes botones de acciones dentro de las mismas para poder crear imagenes interactivas. (como cuando en las paginas de aerolineas permite pinchar encima de un pais para desplegar las opciones de destion o salidas desde ese pais).

Para crear los mapas dentro de las imagenes. Primero hay que insertar una imagen con la etiqueta <img> con todos sus respectivos atributos, despues, dentro de las etiquetas img se inserta la etiqueta <map> para definir las zonas o regiones que va a tener dicha imagen.

La etiqueta <map> tiene un atributo propio llamado "texto". Este atributo sirve para identificar de manera unica dicho elemento. Por ende este atributo debe tener un identificador unico siempre.

Existe otra etiqueta llamada <area>, en la cual vamos a indicar indicar la forma del area mediante coordenadas. La etiqueta <area> cuenta con los siguientes atributos:

Atributos de la etiqueta <area>

  • href = "url" - Indica la Url a la cual se accedera al pinchar sobre dicha area.

  • nohref = "nohref" - Sem emplea para las areas que no son seleccionables.

  • shape = "default | rect | circle | poly" - Indica el tipo de area que se define (toda la imagen | rectangular | circular | poligonal).

  • coords = "lista de numeros" - aqui se declaran las coordenadas para las formas del area. Se declaran con numeros enteros separados por comas. Las areas se declaran de la siguiente forma.

    • Default = toda la imagen.
    • Rectangular = X1,Y1,X2,Y2 (coordenadas X e Y del vértice superior izquierdo y coordenadas X e Y del vértice inferior derecho).
    • Circular = X1,Y1,R (coordenadas X e Y del centro y radio del círculo).
    • Poligonal = X1,Y1,X2,Y2,...,XnYn (coordenadas de los vértices del polígono. Si las últimas coordenadas no son iguales que las primeras, se cierra automáticamente el polígono uniendo ambos vértices)

Para aplicar un mapa a una imagen debemos introducir el atributo "usemap = "nombre_mapa"". usando el nombre que usamos en la variable "name = "nombre_mapa"" de la etiqueta >map>.
