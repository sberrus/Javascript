Tablas En HTML

Veremos todos los aspectos y el uso correcto de las tablas en HTML

En hmtl podemos insertar tablas para ordenar y mostrar la informacion que nosotros deseemos mostrar.

HTML es un lenguaje de marcado de texto, por lo tanto, anteriormente se utilizaban las tablas para estructurar las paginas de HTML. Esto se dejo de hacer debido a que resulta mucho mas eficiente ordenar el aspecto visual de las paginas web a traves de CSS.

Las tablas constan de los tipicos elementos que podemos encontrar en otros softwares o tablas como podrian ser las tablas en excel por citar alguno. Existen filas(row), las columnas se les denominan datos pero tienen la misma naturaleza que las columnas en excel o las cell data En Java. Cabeceras (theader), el pie (tfooter), el cuerpo (body) que cada uno de sus elementos cuentan con unos atributos y caracteristicas especiales para cada uno de los casos en los que vayamos a darle uso a las mismas.

En las tablas de html se implementan utilizando la etiqueta <table> dentro de esta se crea la tabla siendo <tr> la etiqueta que implementa la fila y dentro de esta fila se introduciran el resto table data <tr> para ir rellenando el contenido de dicha celda o dato.

Cada una de las etiquetas tiene su propio atributo. Como explicaremos a continuacion:

Etiqueta <table>

La etiqueta <table> es la encargada de encapsular las tablas que se desean insertar en nuestro documento html. Dentro de las mismas se insertaran las etiquetas <tr>(table row, filas) y <td> (table data, columna). Aunque en HTML no define las columnas como tal, sino que se consideran celdas de datos.
Esta cuenta con los siguientes atributos:

  • sumary = "texto" - Permite describir el contenido de la tabla en cuestion. (Lo utilizan los navegadores para personas discapacitadas)

Etiqueta <tr>

Esta se utiliza para declarar las filas de la tabla. Va creando de arriba hacia abajo las filas que utilizara la tabla en cuestion. Dentro de esta etiqueta se incorporara la etiqueta <td> siendo esta la encargada de mostrar los datos. De igual forma se profundizara mas adelante al respecto. Esta etiqueta no cuenta con ningun atributo.

Etiqueta <td>

La etiqueta <td> es la encargada de los datos que deseamos introducir dentro de la celda de la tabla en cuestion. Esta etiqueta es la mas compleja dentro de las tablas basicas. La etiqueta <td> cuenta con los siguientes atributos:

  • abbr = "texto" - Permite describir el contenido de la celda. (sobre todo para mejorar la accesibilidad de la tabla para las personas discapacitadas.)

  • headers = "lista_de_id" - Indica las celdas que actuan como cabeceras para esta celda (los titulos de las columnas y las filas). Funciona como una ID de las celdas

  • scope = "col, row, colgroup, rowgroup" - indica las celdas para las que esta celda sera su cabecera. Ej: scope="col" indica que esta celda es la cabecera de todas las demás celdas que están en la misma columna.

  • colspan = "numero" - Número de columnas que ocupa esta celda

  • rowspan = "numero" - Número de filas que ocupa esta celda

Etiqueta <th>

Esta Etiqueta se utiliza para definir las celdas que van a ser cabecera de fila o de las columnas.

  • scope = "col, row, colgroup, rowgroup" - indica las celdas para las que esta celda sera su cabecera. Ej: scope="col" indica que esta celda es la cabecera de todas las demás celdas que están en la misma columna.

  • colspan = "numero" - Número de columnas que ocupa esta celda

  • rowspan = "numero" - Número de filas que ocupa esta celda

Etiqueta <caption>

Esta Etiqueta sirve para establecer la leyenda o titulo de la tabla en cuestion para definirlas de entre las que hayan presentes.
