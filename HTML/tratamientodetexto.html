Tratemiento de Texto en HTML

En HTML tenemos diversas formas para tratar el texto cuando se desea hacer cosas como introducir abreviaturas, citar texto de otras fuentes, introducir texto con algunos tipos de tratamiento de resalte etc.. para ello tambien contamos con diversas etiquetas que nos permiten realizar estas acciones. por ejemplo:

Etiquetas em y strong

En este parrafo utilizaremos las etiquetas em para resaltar texto que sea de mediana importancia en un parrafo o cualquier texto convirtiendola en cursiva. Esta etiqueta puede estar dentro de otras etiquetas de texto como la p.
de igual forma contamos con la etiqueta strong esta permite convertir el texto en negrita para resaltar el texto que sea de mayor importancia y que deseen destacar mas que los demas.

Etiquetas de modificacion de texto ins y del

Con las etiquetas ins y del permiten resaltar las modificaciones que se le hayan hecho a algun documento o texto dentro de una pagina web. Estas etiquetas tienen atributos propios para indicar la pagina de donde se ha sacado la informacion para realizar la modificacion y los datos de fecha del mismo. Siendo los el atributo "cite" el usado para introducir la ruta en la pagina web si la hubiere y "date" para indicar numericamente los datos de fecha de la modificacion. Ejemplo:

Texto citado en una pagina web etiqueta blockquote

La etiqueta blockquote es una etiqueta que permite introducir a la pagina web una cita textual de otro sitio de manera que este se vea resaltado. esta etiqueta es de bloque y permite resaltar el texto citado dentro de una pagina. Este tambien cuenta con un atributo cite que permite introducir el sitio web de donde se esta extrayendo la cita.
