Concepto HTML

HTML (Hyper Text Markup Lenguage), es un lenguaje de programacion que permite que los motores de busqueda Web (Buscadores) permitan leer este lenguaje e interpretarlo para que se pueda visualizar de manera grafica a los usuarios.

Concepto de etiquetas.

Las etiquetas son la manera en la podemos ordenar los diferentes componentes dentro del lenguaje html.
las etiquetas funcionan en si como pequeñas cajas en las cuales puedes introducir texto, imagenes, videos, etc...
Hay distintos tipos de etiquetas con las cuales podemos interactuar con HTML. Existen etiquetas que rellenan todo lo que este dentro de una caja, por ejemplo las etiquetas "h1, h2, h3 ... h6, p, etc..." y otras que solo ocupan lo que este dentro de dichas etiquetas "br, b, a, strike, ect...". veremos luego los diferentes tipos de etiquetas con el curso.
Las Etiquetas se tienen que abrir y cerrar mediante signos de mayor y menor. Para abrir una etiqueta se usan los signos y la etiqueta dentro de ellos. ej: < etiqueta > y se cierran agregando un / antes de la etiqueta. ej: < /etiqueta >. estas deben de ir pegadas.
Hay distintos tipos de Etiquetas pero no todas deben cerrar como por ejemplo la etiqueta de salto de linea en texto "br".
