Objeto Window

Uso de JavaScript para eventos del navegador.

Introducción

Window es un objeto de JavaScript con la más alta jerarquia del document. Todo depende del Window. Hace referencia a la ventana del navegador y toda la interacción que hagamos con la misma.

Window tiene una importancia y un sin fin de metodos y funciones además de miles de propiedades, que no se verán todas debido a que la gran mayoria son para realizar ciertas funciones muy especificas que por los momentos no haremos uso de ellas. De igual manera podemos revisar en internet la documentación del objeto Window.

Este objeto hereda las propiedades del evento eventTarget que se explicará más adelante
