Temporizadores en JavaScript

Los temporizadores nos permiten utilizar el tiempo como herramienta de desarrollo y poder implementar con esta multiples tareas de manera eficaz.

setTimeout("funcion","cantidadTiempoEnMiliSegundos")

Con esta herramienta podemos establecer un delay para que se ejecute una función después de haber pasado la cantidad de tiempo enviada como segundo parámetro en milisegundos.

clearTimeout(variable con "setTimeout")

Si queremos que se cancele el setTimeout() para que codifiquemos la cancelación de dicha ejecución. Primero, tenemos que crear una variable que almacenara el temporizador como variable global, de esta manera podemos en otra función indicar mediante el metodo clearTimeout(variable) la cancelación de dicho temporizador.

setInterval(funcion,milisegundos)

Este temporizador nos permite en vez de ejecutar una función pasado el tiempo establecido, está, nos permite ejecutar unma función cada cierto tiempo que nosotros indiquemos como segundo parámetro.
