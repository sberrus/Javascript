Eventos en JavaScript

Con los eventos podemos añadirle dinamísmo a la página web siendo una de las características principales del lenguaje. Puede definirse a "evento" como cualquier cambio que haga la web o el usuario en el navegador.

Eventos del mouse.

  • click: Evento que se ejecuta al darle click a dicho elemento.
  • click: Evento cuando se da doble click.
  • mouseover: cuando se pasa el mouse por encima de un elemento o de sus hijos.
  • mouseout: cuando se sale el mouse del elemento o de alguno de los elementos hijos.
  • contextmenu: se ejecuta la tocar el click derecho.
  • mouseleave: se ejecuta al momento en el que el puntero sale del elemento al que se le ha aplicado
  • mousemove: Se ejecuta al momento de mover el mouse por encima del elemento al que se le aplica.
    Pasa el Mouse por encima de la zona roja para ver el efecto de los eventos "mousemove" y "mouseleave", además de "dblclick"
  • mousedown: este se ejecuta cuando se ejecuta el click izquierdo. La principal diferencia entre "mousedown" y "click", es que con "mousedown" el evento se ejecuta al instante de que el boton izquierdo del mouse realiza el click, a diferencia del evento click que se ejecuta al momento de levantar el dedo del boton izquierdo del mouse. Otra particularidad es que con el evento "click" si levantas el click fuera del elemento que contiene el "listener" este no se ejecuta.
    Deja el click presionado encima de este bloque
  • mouseup: es la contra parte de mousedown, este se ejecuta cuando el click se deja de presionar dentro del elemento al que se le aplica indiferentemente de de que el click se haya hecho fuera del elemento al que se le aplica. Este se ejecuta cuando levantas el click dentro del elemento que contiene el listener
    Has click fuera de este bloque y suelta el click aqui

Eventos del teclado

Los eventos de teclado se ejecutan al momento de que se indique ciertos comportamientos para ciertas teclas del teclado.

  • keydown: este evento se ejecuta cuando cualquier tecla se presiona
  • keypress: Este evento se ejecuta al momento de presionar y luego soltar cualquier tecla.
  • onkeyup: este evento se ejecuta al momento de soltar la tecla y deje de presionarse

Eventos de la interfaz

Estos eventos nos permiten realizar funciones a la hora de que en el navegador o alguno de sus componentes o elementos ocurran algo, como errores, cargas etc...

  • error: este evento se ejecuta al momento de que si algun elemento genera algun error (como que no cargue una imágen completa.), al generarse el error se ejecute una acción.
    ** la idea es que al momento de cancelar la carga de la imágen ver como esta dispara el evento "error" pero no je puede.
  • load: Este evento se aplica al windowObjet indicando que cuando este se cargue del todo se ejecute el evento.
  • beforeunload: este evento también se aplica directamente al objeto window. Ejecuta una acción antes de irse de la página la cual tenga este elemento. Ir a google
  • unload: esta evento se ejecuta al momento de que ya te has ido del sitio
  • resize: este evento se ejecuta al momento de que cambie el tamaño de vista del documento. [abrir consola y cambiar el tamaño de la ventana para ver la ejecucion de este evento]
  • scroll: este evento se ejecuta cuando cada vez que se haga scroll en la ventana a la que se le aplica el evento.

  • selected: este evento se ejecuta al momento de que el usuario haga selec (que subraye algun texto) del elemento al que se le aplica.
