Posibilidades y Limitaciones

Principales características

Las principales limitaciones que tiene JavaScript tienen que ver más con las propias limitaciones de cada navegador, pero principalmente son las siguientes:

  • Los scripts no pueden comunicarse con otros scripts que no pertenezcan al mismo dominio en donde se ejecutan.

  • Los scripts no pueden cerrar ventanas, ni hacer que sean muy pequeñas, ni demasiado grandes; tampoco pueden hacer que las ventanas aparecen en una zona no visible para el usuario (Estas condiciones dependen más del propio navegador).

  • Los scripts no pueden acceder a archivos del usuario y tampoco pueden leer o modificar las preferencias del navegador.

  • Si un script dura demasiado tiempo en segundo plano (ya sea por un error de programación o por otra razón) el navegador informa de que hay algún programa consumiendo recursos en el mismo y la posibilidad de detener su ejecución.

  • A pesar de todo se pueden saltar algunas de las consideraciones previamente vistas. Pero se requiere hacer una firma digital y solicitar ciertos permisos al usuario para hacer uso de estas.

En la actualidad la gran mayoría de navegadores soportan de manera nativa el uso de scripts en JavaScript.

JavaScript en otros entornos

JavaScript permite crear programas para diversos entornos, Adobe Acrobat permite en sus PDF incluir código JavaScript dentro de sus archivos. Photoshop permite introducir pequeños scripts dentro del programa y Java desde la versión 6 introdujo una librería (denominada javax.script) que permite integrar ambos lenguajes.

Por último, aplicaciones como Yahoo! widgets y el dashboard de Apple utilizan JavaScript para programar sus widgets.
