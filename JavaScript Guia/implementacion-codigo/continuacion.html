Nuevas funciones útiles.

prompt(String)

Con esta función podemos pedirle al usuario datos mediante una ventana emergente con un mensaje.

Función .write(String)

Esta función permite escribir directamente codigo HTML donde se le indique. Hay que tener en cuenta de que lo que se introduzca mediante esta funcion es codigo html y podemos introducir etiquetas, atributos y lo que deseemos siempre en código XHTML. Esta función crea un documento XHTML desde 0 así que su uso es recomendable solo para explicar cosas.

Arrays asociatívos

Los arrays asociatívos son aquellos que almacenan información y se accede a ella mediante asociaciones literales. Se trabaja parecido que con los datos JSON, el cual primero se indica la variable y se le asignan los valores con el operador de asignación. Después se introducen campos dentro de este objeto y se le asignan valores. Cada uno de los campos y sus valores se separan mediante comas ","
let pc1 = {
nombre: "Samuel",
apellido: "Berrus",
edad: 25
};

Uso de las acentos inversos (``) y de las variables ${variable} para introducir en las cadenas de texto o variables String.

En javascript además de poder usar las comillas simples ('') y las dobles comillas (""), para introducir cadenas donde se necesite. Pero, también podemos usar los acentos inversos (``) ya que estos nos brindan ciertas funciones que nos facilitaran el evitar conflictos de comillas (cuando queremos utilizar comillas dobles dentor de una variable o simples pero sin que estas cierren la variable), y para introducir variables dentro de estas ya que nos proporcionan una manera más comoda de hacerlo.

Uso de las variables ${variable} dentro de las cadenas.

Con el uso de estas variables podemos introducir en las cadenas de texto que se hayan tipeado dentor de acentos inversos (``), de manera que nos facilita la tarea de introducir variables dentro de las cadenas de texto. En vez de utilizar los signos de concatenacion para introducir una variable:
let texto = "Hola " + nombre + " Como estas? // devuelve "Hola samuel como estas?"
Con el uso de las variables %{variable} se haría mejor de la siguiente manera:
let texto = `Hola ${nombre} como estas?` // devuelve "Hola samuel como estas?
De esta forma el código es más limpio y fácil de mantener.

Uso de las funciones flechas (=>)

Con el uso de las funciones flechas podemos simplificar las funciones ya que reducen la sintaxis de las mismas.

Para hacer uso de las funciones de flechas hay que realizar la siguiente sintaxis: Primero hay que añadir el asignador de variables "const" luego el nombre de la función seguido del operador de asignación, después se añaden los argumentos entre paréntesis, si solo se envia un argumento no hace falta encerrarlo entre paréntesis y si no se envia ningún argumento no hace falta poner los paréntesis tampoco. Luego se añade la flecha (=>) indicando de que esa variable es una función y se abre la sección de instrucción con las llaves {instrucciones}.

const function = (argumentos) => {
[instrucciones]
}

De esta manera podemos trabajar también con funciones.
