Variables y tipos de variables.

Variables

Las variables son un espacio en memoria que utiliza el programa para almacenar información que será luego utilizada dentro del programa.

Las variables pueden ser modificables o inalterables tomando en cuenta las especificaciones del programa. (Parecido a cómo funcionan las variables en Java, pero con la diferencia de que estas tienen un comportamiento diferente al momento de declarar el tipo de variable, esto se verá más a delante).

La palabra reservada “var” se utiliza para declarar una variable por primera vez. Cuando se utiliza la variable en el resto de la instrucción solo hace falta indicar el nombre de la variable.

Cuando una variable se le asigna un valor se dice que ha sido inicializada, en JavaScript no hace falta inicializar las variables con declararlas es suficiente.

Otra de las características de las variables en JavaScript es que no hace falta declararlas, por lo tanto, el uso de la palabra reservada “var” puede omitirse al momento de declarar una variable (aunque no hace falta declarar las variables con la palabra reservada “var”, esta toma relevancia al momento de declarar variables locales y globales, esto se verá más adelante). Al no declarar la variable esta será declarada de manera predeterminada como una variable global.

En cualquier caso, también se recomienda declarar todas las variables principalmente por temas de mantenimiento del código.

El nombre de la variable se considera a su vez como identificador y debe cumplir las siguientes reglas:

  • Solo puede estar formado por letras, números y los símbolos “$” y “_”.
  • El primer carácter no puede ser un número.

Tipos de variables

Otra característica de JavaScript es que el tipo de variable se define al momento de almacenar en ella un valor (texto, número, decimales, etc.), dicha variable toma el tipo del valor que ha almacenado.

Valores númericos

Pueden ser enteros (intenger) o decimales (float). Los valores decimales en JavaScript se declaran con puntos “.” En vez de comas “,”.

Entero = 1
Decimal = 1.1
Error decimal = 1,1

Cadena de Texto

Las cadenas de texto son texto que se almacena en las variables, se declaran asignándoseles con comillas simples o comillas dobles.

Texto con comillas dobles = “ejemplo”
Texto con comillas simples = ‘ejemplo’

Si deseamos introducir comillas simples o dobles en una cadena de texto para que se almacene tal cual debemos alternar las comillas que deseamos que se muestren. Por ejemplo, si deseamos introducir un texto que contenga comillas dobles, la variable se debe declarar con comillas simples, y viceversa.

Cadena con comillas simples internas = “texto ‘con comillas’ simples”.
Cadena con comillas dobles internas = ‘texto “con comillas” dobles’.

Existen también otros caracteres que son difíciles de introducir dentro de las cadenas como (los espacios, las nuevas líneas, etc.) Para resolver estos problemas JavaScript define un mecanismo para solucionar estos problemas. Solo hay que incluir una serie de caracteres para declarar dichos caracteres que queremos utilizar.

Una nueva linea \n
Un tabulador \t
Una comilla simple \'
Una comilla doble \"
Una barra inclinada \\

Arrays

Los Arrays, también conocidos como vectores, matrices o arreglos (Aunque generalmente en programación se conocen en términos generales como arrays) son una colección de datos que se almacenan en una variable. Todas son del mismo tipo y son muy útiles cuando tienes que almacenar datos de una misma naturaleza, pero sin tener que declararlos uno a uno.

En vez de declarar variables de esta manera …

var dia1 = "Lunes"
var dia2 = "Martes"
...
var dia7 = "Domingo";

Para agilizar el proceso y mejorar el mantenimiento se recomienda utilizar un arreglo. Para declarar un arreglo se hace como si fuera una variable simple, con la diferencia que los datos que almacenara deberán ser arropados por corchetes [] y separados por comas “,”.

var dias = ["Lunes", "Martes", "Miércoles", "Jueves", "Viernes", "Sábado", "Domingo"];

De esta manera es más fácil de mantener y de utilizar estos datos que son todos de la misma naturaleza y no tenemos que almacenar uno a uno en una variable distinta. (Los arrays tienen muchas formas de trabajar pudiendo funcionar como pilas, matrices y diversas formas de insertar, extraer y eliminar datos de esta. Pero eso se verá más adelante.).

Para acceder a los datos del array es muy simple, solo hay que poner el nombre de la variable y entre corchetes “[]” indicar el índice de la posición que ocupa dicha variable. (JavaScript es un lenguaje de indexado 0, esto quiere decir que el primer valor de los índices es 0 en lugar de 1). Tomando en cuenta la variable declarada anteriormente que almacena los días de la semana podemos extraer los datos de la siguiente manera.

var diaSeleccionado = dias[0]; // diaSeleccionado = "Lunes"
var otroDia = dias[5]; // otroDia = "Sábado"

Booleanos

Las variables de tipo booleano son variables lógicas. Son utilizadas para la programación avanzada (esto se verá más adelante) y su funcionamiento es realmente sencillo.

Estas variables declaran dos estados, true y false (verdadero y falso). Para declararlos simplemente hay que utilizar los valores (true) y (false) respectivamente.

var clienteRegistrado = false;
var ivaIncluido = true;
