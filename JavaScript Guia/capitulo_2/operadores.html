Operadores en JavaScript

Operadores en JavaScript lógicos, de asignación, de acumulación, etc...

Conceptos Básicos

Los operadores nos permiten modificar los valores de las variables para darles uso y asignarles valores, con estos operadores podemos hacer cálculos complejos y tomar decisiones lógicas en función de comparación y otros tipos de condiciones.

Asignación

Los operadores de asignación son los que nos permiten introducir valores dentro de las variables. La que hemos estado utilizando hasta el momento es el operador de asignación “=” a la izquierda de este operador se pone el nombre de la variable a la que se le va a asignar un valor y a la derecha se le asignan valores, variables, condiciones lógicas, etc.

Incremento o Decremento

Estos dos operadores son solo validos para las variables numéricas y se utilizan para incrementar o decrementar en una unidad el valor de una variable.
Ejemplo: var numero = 5;
++numero;
alert(numero); // numero = 6

El valor de incremento se declara con el uso del nombre de la variable seguido de los caracteres “++”. Y para decrementar lo mismo, pero con los caracteres “--“.

var numero = 5;
--numero;
alert(numero); // numero = 4

El uso de los caracteres de incremento o de decremento en función de su disposición en la asignación a una variable puede alterar la forma en la que el programa realiza dichas acciones.
Al momento de incorporar al código los caracteres de incremento o decremento al inicio de la variable, al momento de usar el programa esta variable realizará el incremento o decremento antes de terminar la función. Si la colocas al final de la variable, el programa realizará el incremento o decremento después de haber culminado la ejecución de la función.

var numero1 = 5;
var numero2 = 2;
numero3 = numero1++ + numero2;
// numero3 = 7, numero1 = 6 var numero1 = 5;
var numero2 = 2;
numero3 = ++numero1 + numero2;
// numero3 = 8, numero1 = 6

Lógicos

Los operadores lógicos son los necesarios para poder realizar operaciones complejas dentro de la aplicación. El resultado de cualquier operación que contenga operadores lógicos dará como resultado un valor booleano.
