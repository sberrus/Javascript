Glosario y sintaxis de JavaScript

Glosario y sintáxis básica de la implementación de JavaScript

Introducción

En esta clase veremos todo lo relacionado con el glosario básico para poder trabajar de manera correcta en JavaScript y su implementación dentro del código.

Glosario Básico

  • Script:

    Cada una de las aplicaciones programas o trozos de código que contengan código JavaScript.

  • Sentencia:

    Una sentencia son cada una de las instrucciones que forman un script.

  • Palabras Reservadas:

    Son las palabras en inglés que se utilizan para crear las sentencias dentro del script. Por lo tanto no puden ser utilizadas libremente a lo largo del script. Las palabras reservadas que utiliza JavaScript son: break, case, catch, continue, default, delete, do, else, finally, for, function, if, in, instanceof, new, return, switch, this, throw, try, typeof, var, void, while, with.

Sintáxis básica

Sin entrar en profundidad en el porqué simplemente iremos a las características generales de la programación en JavaScript, Siendo muy parecidas a las que podemos ver en los lenguajes de programación Java y C. Y dichas características son:

  • No se tienen en cuenta los espacios en blanco ni las nuevas líneas.

    El interprete JavaScript omite todos los espacios en blanco que no esten previamente codificados en le código fuente.

  • Se distinguen las mayúsculas y las minúsculas.

    En JavaScript si se intercambian mayúsculas y minúsculas el programa simplemente no funciona.

  • No se define el tipo de variable

    Los tipos de variables dentro de JavaScript no se declaran como se hace en otros lenguajes como Java o C. Este interpreta el tipo de variable que va a utilizar tomando en cuenta el tipo de dato con el que este trabajando en ese momento. Siendo posible almacenar diferentes tipos de datos en una misma variable.

  • No es necesario terminar cada sentencia con un punto y coma ";"

    Aunque no sea obligatorio si se puede y se recomienda implementarlo, sobretodo para separar sentencias y poder organizar mejor visualmente el código.

  • Se pueden incluir comentarios.

    En JavaScript se pueden usar comentarios de dos formas:

    • Comentarios en línea:

      Se incluyen haciendo uso de dos barras inclinadas a la derecha "//" seguido del comentario en cuestión. Este comentario solo se puede usar en una sola línea de código

    • Comentarios en bloque:

      Estos comentarios son los que nos permiten usar varias líneas como deseemos. También se suele usar para hacer que una o un grupo de sentencias no se ejecute cuando introducimos estas dentro del comentario en cuestión. Para hacer uso de estos comentarios tenemos que indicarlos mediante el uso de los caracteres "/*", indicando el inicio del comentario y "*/", para indicar el final del mismo.
