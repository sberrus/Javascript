Uso de código JavaScript dentro de XHTML

Implementación

En esta clase Veremos como implementar JavaScript dentro de los documentos XHTML. Para introducir código JavaScript disponemos de 2 métodos y 3 maneras distintas. Pero antes de esto tenemos que explicar el uso de la etiqueta <script>.

Uso de la etiqueta script

Esta etiqueta se usa principalmente cuando se implementa dentro del documento o cuando se introduce un código JavaScript enlazado de un documento externo. Para usar la etiqueta <script> es necesario indicar mediante el atributo "type" el tipo de script que es. En este caso se implementaria el siguiente script "text/javascript" para indicarle al documento XHTML que este va a hacer uso de un código JavaScript.

Visto lo relacionado con la etiqueta <script> Veremos como introducir Código JavaScript dentro de los documentos XHTML.

Método de importación de código Javascript desde un archivo externo.

Este método es el más aconsejado debido a que es más sencillo modificar y mantener el código sin tener que cambiar página por página todos los códigos que deseemos modificar.

Para ello tenemos que crear un archivo externo en nuestro editor de código preferido y añadirle al archivo la extensión ".js". Para enlazar el archivo a la página que deseemos tenemos que introducir en el <head> un elemento <script> con su respectivo "type" y ademas un atributo src"/ruta" que indicara la ruta ya sea absoluta o relativa del documento que contiene el código JavaScript que deseamos utilizar.
