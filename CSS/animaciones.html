Animaciones en CSS

Conceptos básicos

Las animaciones en CSS funcionan parecidos a las transiciones pero para poder usar estás, tenemos que hacer uso de la regla @KeyFrames para definirlas dentro del documento CSS.

Uso de las @keyframes

Para usar las @keyframes tenemos que introducir dentro del documento CSS la regla @keyframes, seguido del nombre que le vamos a dar a dicha animación y luego abrir los corchetes "{}" donde introduciremos dentro de estos las instrucciones par dicho @keyframes. EJ: @keyframes desplazar { INSTRUCCION... }

Uso de las propiedades from{} y to{}

Al usar estas propiedades dentro de las instrucciones del @keyframes, indicamos las propiedades de inicio y finales del mismo. Siendo from{} las instrucciones del inicio de la animación y to{} las del final de la animación.

Asignar @keyframe a algún elemento dentro del HTML. Propiedad (animation-name y animation-duration)

Para asignarle la animación a algún elemento HTML tenemos que darle a algún selector al cual se le va a aplicar la animación la propiedad animation-name, indicando como valor el nombre del @keyframe y el valor animation-duration indicando como valor el tiempo que va a tardar la animación en realizar toda la instrucción. Estos dos valores son obligatorios a la hora de trabajar con las animaciones en CSS.

Uso de porcentajes para las @keyframes

Con el uso de los porcentajes dentro de los estados del @keyframe podemos indicar más estados para la animación con la que estemos trabajando. Los estados from{} y to{} equivaldrián a los porcentajes 0% y 100% respectivamente. Pero la principal diferencia entre usar un tipo de valor y otro es que al usar porcentajes podemos dar más de un solo estado dentro de la animación. Podemos usar tantos estados como puntos de porcentajes hay del 0% al 100%.

Propiedad animation-iteration-count

Con esta propiedad podemos indicar la cantidad la cual dicha animación se va a repetir. Este valor tiene que ser númerico. Para agilizar la clase, le hemos aplicado dicho valor a la caja anterior para que se aprecie como se comporta dicha propiedad en las animaciones.

Esta propieadad nos permite repetir la animación tantas veces queramos dandole un valor númerico siendo este la cantidad de veces que se va a repetir la animación.

Valor infinite

Si queremos que este valor se repita de manera infinita o indefinida se usa el valor infinite

animation-direction

Con esta propiedad podemos cambiar el flujo de la animación y hacer que corra de manera inversa a como esta programada.

Valor normal

Con la propiedad con valor normal indicamos el flujo normal de la animación, yendo de from{} a to{} o de 0% a 100% respectivamente.

Valor reverse

Tiene un comportamiento inverso al valor "normal"

Valor alternate

Con este valor podemos alternar el comportamiento de la animación empezando con el valor "normal" y luego alternando con "reverse" creando así, una animación que tenga un comportamiento más fluido.

Propiedad animation-fill-mode

Con esta propiedad podemos indicarle a la animación como ésta debe comportarse al momento de terminar dicha animación (Esto solo sirve para las animaciones pensadas a que no se repitan o se repitan pocas veces, sino no se presenciará la animación.)

Es útil para cuando deseamos que una animación no vuelva a su estado original o valor:none. Qué se puede observar cuando al terminar la animación, el elemento al que se la aplicamos vuelve a su posición inicial.

Valor forwards

Con este valor indicamos que al finalizar la animación, el elemento tenga el aspecto y las instrucciones de el estado final de la misma (to{} o 100%) y que se mantenga allí.

Uso de la propiedad animatioon-fill-mode con valor forward

Valor both

Este valor da más juego para cuando utilizamos junto a este la propiedad animation-delay. Al utilizar esta propiedad y tenemos un estado que modifica el aspecto del elemento con la primera animación, si no tenemos en la propiedad animation-fill-mode el valor both, Esta empezará con el valor que tiene en principio mientras dure el delay. Al usar la propiedad both, indicamos que el aspecto que tenga la propiedad en el to{} o en el 0%(o al valor que hayamos indicado para empezar la animación) se aplique junto al delay.

Propiedad animation-timing-funtion

Con esta propiedad y el valor que veremos a continuación, podemos configurar la velocidad y el como se va a representar la animación utilizando el valor cubic-bezier().

Valor cubic-bezier

Esta es la propiedad que nos permite modificar la interacción de la animación y su velocidad de transición. Este valor se aplica al animation-timing-function seguido de unos parentesis que llevaran dentro de si cuatro parámetros que son los que nos van a permitir modificar la transición.

Con estos parámetros podemos crear animaciones de rebote y de diversos tipos que pueden resultarnos utiles a la hora de crear páginas web más creativas.
