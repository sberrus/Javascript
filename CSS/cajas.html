Método de cajas en CSS

Tipos de cajas

En HTML tenemos dos tipos de cajas que repasaremos a continuación:

Cajas en Linea

Estas son las cajas que solo ocupan el espacio que este considere necesario para mostrar su contenido. Estas cajas si dentro de ellas solo mostraremos una cadena de 3 caracteres, la caja solo ocupara el espacio que abarquen esos 3 caracteres. Las cajas de linea pueden y estan siempre dentro de cajas de bloque. Ejemplo: La etiqueta <a>, Esta solo ocupa el espacio que necesite para mostrar su contenido. Ejemplo de caja en linea de la etiqueta <a>

Cajas en Bloque

Las cajas de Bloque son aquellas que abarcan todo el espacio desde el inicio hasta el final del elemento que los contiene. Estas pueden ser por ejemplo las etiquetas <p> <h1> <h2> etc...

Modificaciñon de comportamientos (display)

Para modificar el comportamiento de un elemento en bloque para que se comporte como un elemento en linea y viceversa hay que utilizar la regla CSS (display).
