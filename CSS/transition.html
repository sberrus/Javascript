Transition en CSS

Conceptos Básicos

Transition es una propiedad que nos permite darle transiciones a los aspectos de los elementos dentro de HTML. Es modificar como y en cuanto tiempo los frames de cambio de aspecto se representan en el documento.

Propiedad transition-property: property/ies -all -none

Esta propiedad señala a que elemento o elementos se le va a aplicar la transición se puede indicar directamente el valor "all" para indicar que se le aplica a todos los elementos al que se le aplica dicha transicion.

Propiedad transition-duration: duration/s

Esta propiedad indica el tiempo que va a tardar dicho contenido en cambiar del estado 1 al estado 2.

Estas dos propiedades (property y duration) son obligatorias para poder utilizar dichas propiedades en un elemento. De lo contrario el efecto no se producira.

Combinación de valores y propiedades
