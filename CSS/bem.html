Metodología BEM

Introducción

La metodología BEM consta de unas prácticas las cuales nos permite establecer de manera eficaz el uso de las clases("class") de HTML para mejorar la implementación de las hojas de estilo y asi evitar conflictos. Ejemplo:

El uso de la metodología BEM consta de utilizar clases en los elementos que vayamos a utilizar pero de la siguiente manera. En el bloque de texto anterior con el borde "rojo" vemos que hay un contenedor <div> y dentro de este hay un formulario <form> con 2 elementos <input>. Para aplicar la Metodología BEM hay que darle una clase al contenedor por ejemplo "contact-form" y como los inputs que queremos modificar estan dentro del contenedor <div> de clase "contact-form" pues a los inputs se les asigna una clase empezando con el primer contenedor "contact-form" dos guiones bajos "__" y la clase del elemento "input" en este caso y daria como resultado lo siguiente "contact-form__input" de esta manera le asignamos una clase mas precisa a los elementos que queremos modificar.

En el caso anterior funciona correctamente porque tenemos una misma clase para todos los elementos "input" dentro del elemento "contact-form" pero, ¿que pasa si por ejemplo tenemos 3 elementos pero queremos modificar uno de estos?. Para esto tenemos 2 maneras para realizar dicha tarea:

  1. :first-child

Algo que hay que recordar es que al momento de usar los "__" se utilizan para aquellos elementos que sean de bloque y los "--" para aquellos que sean en linea. Por ejemplo: que ocurre si queremos introducir un elemento h2 dentro de un elemento p que a su vez este dentro de un elemento div. El resultado seria el siguiente: Primero asignamos la clase al contenedor (bloque)"contact-form", después introducimos los "__" ya que el contenedor div es un elemento en bloque. A continuación, introducimos el elemento p de la siguiente manera "contact-form__p" indicando que el elemento p esta dentro del contenedor div. Y por último introducimos "--" y el elemento h2 resultando "contact-form__p--h2" indicando que el elemento en linea h2, esta dentro del elemento en linea p, que a su vez esta dentro del elemento en bloque div.
