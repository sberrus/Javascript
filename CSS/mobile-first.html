Responsive Design y Mobile-First

Responsive Design

Se refiere a responsive design a la capacidad que tiene una página web diseñada principalmente para pantallas de ordenador en adaptarse a las distintas pantallas, ya sean tablets, mobiles u otros dispositivos.

Mobile-First

Este concepto es parecido al anterior pero de manera contraria. Este hace referencia a aquellas páginas webs que son diseñadas principalmente para pantallas de smartphones y su capacidad para que se adapten a pantallas de ordenador. Esto vino después de que Google mejorara el posicionamioento para las páginas que contengan esta particularidad.

Para que HTML pueda interepretar que el contenido que va a mostrar en un smartphone hay que hacer 2 cosas:

1- Agregar Viewport

Primero hay que indicar mediante meta datos que esa pagina tienen disponible una versión para smartphones. Esto se implementando la siguiente linea de codigo en el <head> del documento en cuestión.
<meta name="viewport" content="width=device-width, user-scalable=no">

2- Agregar @media en el codigo CSS.

Esta linea de codigo CSS se utiliza para modificar ciertos aspectos de la página para que esta modifique sus dimensiones y su interacción al momento de que le tamaño sea inferior a los campos que nosotros hayamos indicado mediante la instrucción @media. (Esto es más un concepto responsive design asi que se vera más adelante).
