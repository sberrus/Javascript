Medidas en CSS

Antes de ver lo el resto de partes de este curso veremos lo correspondiente a las medidas dentro de CSS para la estetica de los distintos elementos de HTML.

Unidades de medida fija

Las unidades de medida fija son aquellas que permiten modificar los atributos estéticos de un elemento HTML para modificar su tamaño o dimensiones. Hay diferentes tipos de medidas fijas que se utilizan dentro de CSS como los pixeles (px), los centrimetros(cm), los milimetros(mm) y los puntos(pt).
Como ejemplo utilizaremos la instrucción "font-size" para modificar el tamaño de las letras de un elemento HTML.
La instrucción indica primero el número de unidades que va a utilizar la instrucción seguido de su tipo de unidad. Por ejemplo, la instrucción "font-size:10px;", indica que cada letra debera tener un tamaño de 10px cada una. El principal inconveniente de los elementos de tipo fijo es que son fijos, valga la redundancia. Al usar este tipo de elementos hace complicado que estos elementos se adapten correctamente a los distintos dispositivos ya que no todos tienen el mismo tamaño de pantalla ni la misma resolución. Por ende este tipo de unidades de medida se utilizan principalmente para aquellos elementos que se desea que siempre tengan un aspecto determinado sin importar el tamaño de la pantalla donde estos seran impresos.
Para evitar estos problemas se utilizan las unidades de medida relativas.

Unidades Relativas

Las unidades relativas son aquellas que toman en cuenta los elementos que las contienen para mostrar sus contenidos. Estas medidas son las mas utilizadas ya que permiten adaptar las dimensiones de los elementos para que estos se adapten a las pantallas de los distintos dispositivos que existen.
Como se ha dicho las unidades relativas toman como referencia propiedades de los elementos que las contienen para mostrar los contenidos en pantalla.
Las medidas relativas son: los porcentajes (%), las medidas em, entre otras. Para calcular las medidas relativas, como ya hemos explicado antes, primero hay que saber las propiedades de los elementos contenedores del elemento que vamos a programar. Para el siguiente ejemplo vamos a crear un contenedor "div" para introducir dentro de este elementos con medidas relativas. Y otro elemento div con la fuente más grande para que se vea la demostración.
Hay que tomar en cuenta que por defecto en HTML el tamaño de fuente general para los textos es de 18px. Por lo tanto en los navegadores de ordenadores las letras tendran este tamaño estandar. También hay que saber que vamos a utilizar los mismos valores en ambos contenedores. Ejemplo:

Contenedor Standard

1em 2em 3em 4em

Contenedor con font-style="20px"

1em 2em 3em 4em

La medida em utiliza el espacio que ocupa la letra "m" para declarar el tamaño de los elementos que queremos modificar. Por ende, para calcular el tamaño de los elementos que queremos modificar con este atributo hay que tomar en cuenta primero el tamaño de la fuente del contenedor (Sí esta no esta establecida se utiliza su tamaño estandar "18px"). Después, se multiplica por la medida em en cuestión. Por lo tanto para un elemento que tenga una medida de 3em en un contenedor estándar se haria la siguiente función: 18px*3em=54px.
