Tipos de selectores

En html contamos con varios tipos de selectores con la cual podemos trabajar. Ejemplo:

  • Selector Universal "*"

    Este selector se encarga de modificar el aspecto de todos los elementos de la pagina HTML. Ejemplo:
    * {color:red} Todos los elementos de la pagina tendran color de texto rojo

  • Selector de etiqueta

    Este se encarga de modificar el aspecto de las etiquetas que esten dentro del selector. Ejemplo:
    h1 {color:blue}

    Todas las etiquetas h1 estaran de color azul

    En los selectores tambien podemos introducir tantas etiquetas como nos sea necesario. Ejemplo:
    h1, h2, h3, p {color:green}.

  • Selector por clases

    En html tambien podemos utilizar selectores que modifiquen el aspecto de los elementos que contengan una clase con un valor especifico. Esto lo logramos utilizando como selector el valor de la clase que deseamos modificar con un punto antes de la misma. Ejemplo:

    .clase{color:orange}

  • Selector por ID

    Tambien podemos utilizar el atributo id para cambiar el aspecto de los elementos en hmtl. Funciona similar a los selectores de clase pero utilizando el signo "#" en vez de un punto. Ejemplo: #id {color:grey}

Selectores Especiales

Con los selectores basicos es posible crear cualquier pagina web que deseemos. Aunque tambien podemos utilizar selectores especiales que nos permiten realizar acciones mas avanzadas dentro de CSS como veremos a continuación.

  • Selectores de hijos ">"

    Este selectore permite modificar los elementos que sean hijos directos de un padre en html. se utiliza el simbolo ">" para declarar el selector. Ejemplo:
    p > a {color red}
    Solo se modificara el aspecto a los elementos con etiqueta <a> que esten dentro de una etiqueta de tipo <p> y que sea hijo directo. Sí antes de la etiqueta <a> hay otra etiqueta a esta no se le modificara el aspecto

  • Selector adyacente

    Con este selector podemos modificar el valor de un elemento que se declare adyacente de otro elemento que declaremos en el selector. Estos se declaran con el signo + para definir la adyacnecia. Ejemplo:
    p + p{color:red;}
    En este ejemplo solo se modificara el aspecto de las etiquetas <p> que preceda a otro elemento <p>

Psuedo selectores

Los pseudo selectores son aquellos que indican que modifique el aspecto de algun elemento despues de haberse realizado alguna accion previa, como por ejemplo, que al pasar el mouse por encima de algun elemento este midifique su aspecto. Estos se le añaden a algun elemento del selector seguido de un ":" y el pseudoselector correspondiente. Por ejemplo:

:hover
