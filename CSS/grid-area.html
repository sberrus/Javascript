Grid area

Conceptos básicos

Un área es un conjunto de celdas (preferiblemente consecutivas) que estan dentro de un grid. Para hacer uso de areas tenemos que usar la propiedad grid-template-area: después, entre comillas, vamos a indicar fila por fila las áreas que ocupara cada una de las filas. En el siguiente ejemplo la distribución de las áreas es la siguiente "header header header" "aside main main" "aside main main" "footer footer footer"

Ya asignados las areas en el template luego a cada uno de los contenedores internos del grid-container se les asigna una de las áreas con la propiedad grid-area. Después de asignarle los aspectos deseados tenemos una estructura parecida a la que veremos a continuación.
