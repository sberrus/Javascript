Pseudo-clases en CSS

La pseudo clase funciona a nivel de codigo igual que un pseudo-elemento. Este se aplica directamente a otro elemento y este resulta ser hijo del elemento al que se le aplica.

La principal propiedad de las pseudo-clases es la posibilidad de escuchar eventos para luego realizar sus respectivos cambios (pasar el cursor por encima de el elemento por ejemplo). Este elemento realizar una "escúcha" a continaución veremos algunas pseudo-clases.

Posiciona el cursor por encima del la siguiente caja

Con esta propiedad podemos aplicar diversas modificaciones estéticas a los elementos a la que se la apliquemos que se apliquen al momento de posicionar el cursor por encima de dicho elemento con la pseudo-clase :hover

Con la propiedad transition modificamos la velocidad en la que los elementos cambiaran su apariencia expresada en segundos.

:link y :visited

Con estas pseudo-clases podemos cambiar el aspecto de los links "<a>" antes (:link) y después (:visited) de haberlos visitados.

:active

Esta pseudo-clase funciona al momento de hacerle click a un elemento este cambia su aspecto, pero este se mantendra la misma cantidad de tiempo que la del usuario mientras mantenga presionado el click

Deja el click pulsado aqui para activar la pseudo-clase

:focus

Esta pseudo-clase se aplica principalmente a los elementos de formulario input. Esta pseudo-clase se aplica al momento de que se este enfocando (cuando se le da clic al elemento input como textarea o en los passwords fields, cuando se esta escribiendo algo en dichos campos).

:lang

Esta Pseudo-clase se encuentra en desuso.

Esta pseudo-clase se utiliza para modificar el aspecto de aquellos elementos que contengan la etiqueta de lenguaje lang(es-en-in...etc). Al momento de que a algun elemento le pongamos el atributo "lang" la pseudo-clase busca el valor de dicho atributo mediante el parametro que estara encerrado entre parentesis del pseudo-elemento.

Con el uso de las pseudo-clases podemos realizar diseños para las páginas web dinámicos que resulten más fluidos para la interacción del usuario con la página web. Ádemas, Estas clases permiten generar una sensación de fluidez en la estética de la pagina haciendola más compleja.
