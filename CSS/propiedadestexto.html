Propiedades de Texto en CSS

En esta clase veremos las propiedades de texto que tenemos y podemos modificar en CSS.

Propiedades Comunes

Entre las diferentes propiedades de texto comunes en CSS tenemos las siguientes:

  • font-size

    Esta se encarga de modificar las dimensiones de las letras. EJ:
    1.5em
    12px
    15pt
    2cm
    25mm

  • font-family

    Esta instrucción se encarga de establecer el tipo de letra que deseamos introducir en CSS. ej:

    Georgia
    Arial
    monospace
    cursive
    etc...

    Ir al siguiente enlace si se desea ahondar más en el tema de las Fuentes

  • line-height

    Esta instrucción permite cambiar el tamaño del interlineado de las cadenas de texto.

    line heigth 1
    line heigth 2
    line heigth 3
    line heigth 4

  • font-weigth

    Este elemento se encarga de modificar el tamaño del trazo de la letra. De que se vea mas gorda o mas delgada. EJ:

    WEIGHT 150
    WEIGHT 250
    WEIGHT 350
    WEIGHT 800

    En este caso no se puede visualizar correctamente el uso de esta instrucción porque algunas fuentes no permiten modificar el weight de su fuente pero algunos otras si.
