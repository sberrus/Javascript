Método de cajas en CSS

Normalize

Los navegadores siempre tienen un estilo y unas propiedades por defecto que varian dependiendo del navegador que el usuario final vaya a utilizar. Sabiendo que no todos los navegadores son iguales y tienen las mismas propiedades se creo el Normalize.

Normalize es una solucion interesante para solucionar el problema de las propiedades de los distintos navegadores. Este consta de un archivo CSS que contiene unas instrucciones estandarizadas para que las paginas web se vean de manera correcta en cualquier navegador. Sumado a otras instrucciones que implementamos de manera manual dentro de el archivo Normalize.css podemos usar un estandar para poder programar en CSS de manera correcta sin inconvenientes de los diferentes navegadores presentes.

Para usar el Normalize hay que entrar en la pagina de Normalize, después de esto, se puede implementar de dos maneras:

  • Descargando el archivo CSS

    Desde la página anteriormente mencionada, se descarga el archivo CSS y se implementa en el documento en el cual estemos trabajando. El documento lo puedes descargar en el siguiente enlace

  • Importando el archivo CSS mediante URL

    También podemos implementar el archivo CSS mediante la URL de la pagina en cuestión para implementarlo en el HEAD de nuestra página.

Siempre es recomendable usar los archivos de manera local ya que resulta mas conveniente hacerlo de esta manera para que no haya errores ni problemas a la hora de importarlo mediante URL.
