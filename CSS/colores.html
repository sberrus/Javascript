Colores en CSS

Los colores ya se han utilizado varias veces a lo largo de estas clases pero básicamente es eso los diferentes tipos de colores que podemos aplicar a los elementos y a ciertas propiedades en CSS.

Color Name

Los color name son los colores que podemos utilizar como valor directamente utilizando su nombre tenemos varios color name.

El principal inconveniente de usar este método es que cada navegador tiene definidos sus propios colores. Por ejemplo: Si definimos como valor "red" no va a ser el mismo tono de rojo en mozilla que en firefox por ejemplo.

Lista de colores (color name) en CSS.

Hexadecimal

Para solucionar el problema anteriormente mencionado con respecto a los diferentes colores por navegador existen varios métodos o soluciones para esto. Por ejemplo el valor hexadecimal.

el valor hexadecimal se diferencia porque contiene una almoadilla o signo gato (#), seguido del código del color que deseamos implementar al elemento. Con este valor y con los que veremos a continuación se soluciona el problema de compatibilidad de colores que tenemos con los valores (color name).

RGB, RGB/A HSL Y DEMÁS

Hay diversos formátos para trabajar los colores en CSS. Pero para indagar más al respecto recomiendo visitar la página de Uniwebsidad para saber los demás formátos para la implementación de los colores en nuestros documentos.

El valor interesante es el RGBA ya que este funciona igual que el RGB con al diferencia que este tiene un 4to valor que va a ser igual a que si hayamos implementado una opacity al elemento al que le queremos aplicar dicho color.
