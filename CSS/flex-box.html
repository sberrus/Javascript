Flexbox

Conceptos básicos

Esta es una nueva forma de maquetar páginas web sustituyendo a los anteriores formatos como podian ser el maquetado mediante etiquetas <table>. Después, se empezo a utilizar los maquetados mediante cajas con position float y divs pero tambien surgian ciertos problemas. Ahora para solucionar los problemas anteriores que conllevaban utilizar las anteriores maquetaciones se empezo a utilizar los maquetados Flex-box y Grid.

Para implementar esta maquetación, hay que tener dos elementos en cuenta.

Flex Container

El container es el que contiene a los elementos y dentro de estos seran los que tendran los diferentes elementos del mismo.

El flex conteiner tiene 2 ejes. El main axis(Eje horizontal) y el cross axis (Eje vertical)

Propiedades en flexbox

Las propiedades flex box son las que nos permiten realizar modificaciones a los items que las contiene. Siempre se les dan al flex container.

A los flex items hay que darles altura y anchura para poder visualizarlos si no contienen ningun contenido dentro de estas. Si solo estableces la altura o solo la anchura estos no se veran en el flex container.

Propiedad flex-direction:

Con esta propiedad podemos modificar la dirección la cual los flex containers muestran el flujo de sus contenidos.

Valor: row y row-reverse

Con estos valores indicamos el flujo horizontal (main axis) el cual el flex container va a mostrar los contenidos. El valor row india en el main axis que muestre los contenidos de izquierda a derecha y el row-reverse indica lo contrario, de derecha a izquierda.

Propiedad flex-wrap

Con la propiedad flex-wrap podemos establecer un comportamiento de las cajas en funcion de las dimensiones del objeto que las contenga; si deseamos que mantenga unas dimensiones fijas, si queremos que esta se adapte al ancho y alto de la caja que las contiene etc...

Valor wrap

Con este valor establecemos que la caja indiferentemente del tamaño de el contenedor mantenga siempre las mismas dimensiones.

Shorthand flex-flow

La propiedad flex-flow es un shorthand que nos permite indicar en una misma linea de codigo tanto la propiedad flex-wrap y flex-direction.

Para implementarla hay que utilizar la propiedad "flex-flow" seguido de sus dos puntos y como valores primero los pertenecientes a la propiedad "flex-direction" y luego la propiedad "flex-wrap". Ejemplo: flex-flow: column-reverse wrap;

Alineación entre Flex-Items(Main Axis)

Otra de las ventajas que tienen los flex-box es la facilidad con la que podemos indicarle al programa la alineación y distribución de los contenidos dentro de los flex-containers. Podemos indicar la separación entre las cajas entre otras cosas que veremos a continuación.

Propiedad justity-content

Con esta propiedad podemos modificar la forma en la que apareceran los contenidos dentro de los flex-box. De esta manera podemos indicar que si queremos que las cajas aparezcan centradas, de derecha a izquierda, la distancia entre las cajas, que tanta distancia tiene que tener una caja con relación a la otra, etc...

Alineación entre Flex-Items(Cross Axis)

Con los flex containers tambien podemos modificar el flujo de los flex-items de manera vertical siendo también un gran ventaja a la hora de maquetar los contenidos de las páginas webs.

Propiedad align-item

Con esta propiedad modificamos la distribución y la separación la cual los elementos apareceran dentro del flex container.

Es importante recalcar que esta propiedad va a afectar a la primera fila del flex-container, por lo tanto, si deseamos introducir un elemento que tendra más de una fila es mas util que utilizemos la propiedad "content-align" que veremos más adelante.

Valor center

Una de las grandes ventajas que trae el uso de los flex containers es la facilidad con la que estos nos permiten centrar los elementos de manera vertical. Solo con este valor los flex items se van a alinear de manera vertical justo en el centro. Cosa que anteriormente era más complicado de lograr.

center

En el cuadro anterior se puede observar como los flex-items estan alineados el cross axis justo en el centro de el contenedor.

Valor flex-end

Con este valor indicamos que distribuya los items desde abajo hacia arriba.

flex end

flex start

Con esta propiedad indicamos que los items que no contengan la propiedad height y width mantengan la posicion del item más grande pero sin ocupar el resto del espacio disponible de la caja.

flex-start

Como se pueden observar las cajas solo ocupan el espacio necesario que estas necesita para aparecer en el container pero sin ocupar más espacio de la cuenta.

Valor stretch

Con este valor indicamos que los flex items ocupen todo el espacio vertical disponible en la caja indiferentemente del tamaño del contenido que contengan.

stretch

Propiedad content-align

Esta propiedad sirve para modificar el flujo de los flex-items y como se van a distribuir dentro del flex-container correspondiente. Esta propiedad si afecta al total de flex-items sin importar que tantas filas contenga el mismo.

Importante destacar que algunos elementos deben tener obligatoriamente la propiedad flex-wrap para que surja efecto.
