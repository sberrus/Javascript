Pseudo-elementos en CSS

Es un elemento que no forma parte del DOM pero si altera el comportamiento de algunos elementos que si forman parte de este. Su uso es meramente estético.

::firstline

Antes que nada hay que saber que solo funciona con los elementos de bloque. Como indica su nombre altera el comportamiento de la primera linea del bloque al que se le aplique el pseudo-elemento. Este pseudoelemento se le aplica al elemento al que deseamos aplicarle esta propiedad indicando después de este seguido de dos dobles puntos (::) seguido del pseudoelemento (::first-line).

::placeholder

El placeholder se aplica a los elementos de formulario input. El cual al elemento se le puede dar una propiedad llamada "placeholder" que es la que muestra un texto que no forma parte de los valores del formulario. Generalmente este forma parte de las descripciones de las mismas.

Con el pseudo-elemento ::placeholder podemos modificar el aspecto de este elemento.

En en anterior ejemplo podemos observar 2 elementos de formulario input; uno sin modificar y el otro de color rojo.

::selection

Con este pseudo-elemento podemos modificar el aspecto de (cuando dejamos el clic sobre el texto y seleccionamos parte de alguna cadena).

Para poder observar correctamente este pseudo-elemento tendras que seleccionar (dejar el clic sobre el texto que deseas copiar por ejemplo) para poder mostrar el comportamiento.

Puedes utilizar este mismo texto como ejemplo o el formulario que aparece a continuación.

Tanto el texto pequeño que aparece encima del formulario, como el texto dentro de el textfield tienen establecido el pseudo-elemento selection. Dichos elementos deberian mostrarse de fondo negro con las letras rojas.

::before y ::after

Este pseudo-elemento se aplica como un elemento hijo del elemento al que se le esta aplicando dicha pseudo-elemento. Este a su vez se comporta como un elemento en linea por lo cual no podemos modificar sus dimensiones más allá que las que este hereda del elemento padre.

. Esto es lo que está en la etiqueta <b> dentro del documento HTML.

En el texto anterior se puede ver como se comportan el pseudo-elemento ::before y ::after. En el texto anterior la cadena que esta de color naranja es aquello que forma parte del pseudo-elemento ::before. Su contenido se encuentra en el documento CSS y no forma parte del DOM, se visualiza pero no forman parte del mismo. Este comparte las mismas características que el pseudo-elemento ::after estando de color verde. En el ejemplo anterior.

El contenido de color rojo es aquel que si se encuentra dentro del documento HTML. Como bien lo indica el nombre del pseudo-elemento, lo que este dentro de ::before se muestra antes que el elemento al que se le aplica dicho pseudo-elemento. Y lo mismo con el pseudo-elemento ::after mostrandose después del mismo.

El texto que esta dentro de ambos pseudo-elementos, no es posible seleccionarlo.
