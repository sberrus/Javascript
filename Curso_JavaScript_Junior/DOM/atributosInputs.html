Uso del atributo form

Para observar el comportamiento de dicho atributo tienes que rellenar los campos anteriores y darles algun valor, luego, tienes que darle primero al "botón dentro del form" para que puedas ver que el contenido se va y la página realiza una carga.

Si realizas el mismo procedimiento pero dandole al segundo botón verás que el texto se mantiene y no se realiza ningún proceso de carga porque en sí, la página no está enviando nada.
