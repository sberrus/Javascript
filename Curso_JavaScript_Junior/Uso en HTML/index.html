Arrays asociatívos

Los arrays asociatívos son aquellos que almacenan información y se accede a ella mediante asociaciones literales. Se trabaja parecido que con los datos JSON, el cual primero se indica la variable y se le asignan los valores con el operador de asignación. Después se introducen campos dentro de este objeto y se le asignan valores. Cada uno de los campos y sus valores se separan mediante comas ","
let pc1 = {
nombre: "Samuel",
apellido: "Berrus",
edad: 25
};

Para acceder a los datos de los arreglos asociativos hay que hacer llamadas mediante el campo que aloja el dato. Tomando como ejemplo el arreglo anterior, si queremos extraer la información del campo nombre hay que extraerlo de la siguiente manera =
pc1["nombre"] // devuelve samuel
En el código anterior devolvera el valor que esta dentro del campo "nombre".
