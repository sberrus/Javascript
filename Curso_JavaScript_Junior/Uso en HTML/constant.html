Funciones y tipos de funciones.

Metodo para llamar a los function mediante flecha ( => )

Este es otro metodo para crear funciones. Consta de crear una variable "const" a la que se le asignara la función. La sintáxis es primero la palabra reservada "constant" debido a que esta no se modificara en el futuro, después el nombre de la función seguido del signo de asignación ( = ) entre paréntesis van los argumentos de la función y luego se abren las llaves de las intrucciones.

const slaudar = (nombre) =>{
let frase = `Hola como estas $nombre`;
document.write(frase);
}
