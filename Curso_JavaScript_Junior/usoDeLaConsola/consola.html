Uso de la consola en JavaScript

Con el uso de la consola, la cual podemos acceder mediante el navegador con las herramientas de desarrollador o mediante algunos IDE de JavaScript, podemos hacer multiples cosas que nos ayudaran a mejorar y encontrar errores y bugs dentro de nuestro sistema.

La consola básicamente nos permite insertar código JavaScript lienalmente dentro del documento. Podemos definir una variable "let name = 'samuel'" y usar la función alert() para que nos la muestre en pantalla. "alert(name)" // devuelve samuel.

Podemos modificar el DOM del html o internamente utilizar la consola mediante comandos para la misma por ejemplo con el uso de la función del objeto console .log(), para mostrar contenido dentro de la consola en cuestión. console.log(name) // devuelve en consola "samuel".

El resto de las funciones y códigos se verán en el JavaScript consola.js para que se vea internamente en código el uso de la misma.
